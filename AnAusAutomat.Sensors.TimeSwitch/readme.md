# TimeSwitch
Turn on or off at a specific time.

## Status

#### PowerOn
At configured time.

#### PowerOff
At configured time.

#### Undefined
Not supported.

## Parameters

### Global
| Name | Values | Default | Multiple | Example | Description |
| ------ | ------ | ------ | ------ | ------ | ------ |
| | | | |

### Socket
| Name | Values | Default | Multiple | Example | Description |
| ------ | ------ | ------ | ------ | ------ | ------ |
| On | HH:mm:ss | no | yes | 17:45:00 | Turn on time. |
| Off | HH:mm:ss | no | yes | 23:15:00 | Turn off time. |
